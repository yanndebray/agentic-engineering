# Agentic Engineering

A collection of projects demonstrating agentic AI workflows for engineering applications, using Claude Code with MATLAB, Simulink, and cloud infrastructure.

## Projects

### [autocloud](autocloud/) — Automotive Cloud Data & Virtual Vehicle Simulation

End-to-end pipeline combining real-world fleet driving data (VED dataset, 383 vehicles) from AWS S3 with virtual vehicle simulation using Powertrain Blockset. Covers data loading, EV model building with Virtual Vehicle Composer, parallel simulation with `parsim`, and energy/range analysis.

**Stack:** MATLAB, Simulink, Powertrain Blockset, AWS S3, Parallel Computing Toolbox

### [nasa-aircraft-big-data](nasa-aircraft-big-data/) — NASA Aircraft Sensor Data Engineering

Big data engineering and analysis using aircraft sensor data provided by NASA. Supports predictive maintenance, digital twins, AI with signals, and fleet analytics workflows.

**Stack:** MATLAB, Parallel Computing Toolbox, Cloud Center
